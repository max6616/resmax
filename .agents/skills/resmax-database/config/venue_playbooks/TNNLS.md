# TNNLS Venue Playbook

## 基本信息

- 全称: IEEE Transactions on Neural Networks and Learning Systems
- 出版商: IEEE Computational Intelligence Society
- ISSN: 2162-237X (print), 2162-2388 (online)
- 类型: 期刊（月刊）
- DBLP key: journals/tnn
- OpenAlex source ID: S4210175523

## 数据源

- Primary: OpenAlex API (`openalex_works` parser)
- OpenAlex 提供完整摘要（abstract_inverted_index 格式）
- 每年约 842 篇论文

## 摘要获取

OpenAlex 直接提供，无需额外补全。

## 评审数据

不公开。标记 `review_available=no`。

## 已知经验

- DBLP key 是 `journals/tnn`（沿用旧名 IEEE TNN 的缩写）
- 论文量较大，OpenAlex cursor paging 约需 5 页
