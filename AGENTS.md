# Repository Instructions

This repository contains agent skills and skill tests.

General rules:
- Do not edit files under `skill-tests/runs/` unless explicitly instructed.
- Skill source files live under `.agents/skills/`.
- Test cases live under `skill-tests/cases/`.
- Generated test artifacts must go under a fresh `skill-tests/runs/<run-id>/` directory.
- Do not treat a previous run artifact as evidence for a new run.
